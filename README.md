#  Job interviews App

Stack:

- сборщик - vite
- фреймворк - vue 3 (composition api)
- язык - typescript
- стейт менеджер - pinia
- библиотеки компонентов и стилей - primevue/primeflex/primeicons
- авторизация и база данных - firebase


### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
